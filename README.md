# IAP

### Node.js with Express.js

#### From Analog to Mongoose(use methods instead queries)

| Concept    | SQL         | Mongoose            |
| ---------- | ----------- | ------------------- |
| Insert     | INSERT INTO | create()            |
| Read       | SELECT      | find()              |
| One record | WHERE id    | findById()          |
| Update     | UPDATE SET  | findByIdAndUpdate() |
| Delete     | DELETE      | findByIdAndDelete() |
| Sorting    | ORDER BY    | sort()              |
| Join       | JOIN        | populate()          |
